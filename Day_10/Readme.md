## Key Terms.

Database → Holds collections

Collection → Like a table (stores documents)

Document → BSON-style object

🄚 _id →  Auto-generated, unique identifier

~~~
## 🧠 Query Operators
 * find() 
 * project
 * sort
 * $gt -> greater than
 * $lt -> less than
 * $eq -> equal
 * $ne -> not equal
 * $in -> matches any in array
 * $nin -> doesn't matches any in array
